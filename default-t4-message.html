Welcome to the TERMINALFOUR Site Manager. To use the system, please choose an option from the menu above

The Content menu enables users to add/modify and delete content, as well as manage the structure of the site.

The Rights & Roles menu is used to control user access to the system, both on an individual and a group basis.

The Tools menu gives administrators access to tools for managing the list of languages available, published channels, template and workflow management, amongst others.

The Help menu gives access to the Site Manager help system.
